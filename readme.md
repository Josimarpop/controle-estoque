#Controle de Estoque
Projeto desenvolvido com o objetivo de utilizar o framework Laravel para programação da linguagem PHP.

## Informações do Projeto
Controle de estoque com processos básicos, fazendo operações de inserção, edição e exclusão de produtos de um estoque, com validações em campos que devem ser obrigatórios.

Não foi desenvolvido o gerenciamento de usuários e login do sistema.
